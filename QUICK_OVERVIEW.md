## 🎯 Что это?

**QR Menu SaaS** - готовая SaaS-платформа для ресторанов с QR-меню и заказами через WhatsApp.

```
✅ Полностью готовый MVP
✅ 5000+ строк кода

✅ Production Ready
✅ MIT License
```

---

## 🚀 Запуск за 5 минут

```powershell
# 1. Клонировать
git clone https://github.com/yourusername/qr-menu-saas.git
cd qr-menu-saas

# 2. Установить
npm run install:all

# 3. Настроить .env (см. QUICKSTART.md)

# 4. Запустить
npm run dev
```

**Готово!** Откройте http://localhost:5173

---

## ✨ Основные возможности

### Для ресторанов

- 🏪 Регистрация за 2 минуты
- 🍽️ Управление меню (категории, блюда, фото)
- 🎨 Баннер-слайдер
- 💬 Заказы через WhatsApp
- 📱 Уникальный QR-код
- 🆓 7 дней бесплатно

### Для клиентов

- 📱 Красивое меню с фото
- 🛒 Корзина с модификаторами
- 💬 Заказ одной кнопкой
- 📲 Работает на всех устройствах

### Для админов

- 👨‍💼 Управление ресторанами
- 💳 Управление подписками
- 📊 Статистика платформы

## 🛠️ Технологии

**Frontend:** React 18 • Vite • TailwindCSS • Zustand  
**Backend:** Node.js • Express • Prisma • PostgreSQL  
**Деплой:** Vercel • Render • Supabase (бесплатно!)
