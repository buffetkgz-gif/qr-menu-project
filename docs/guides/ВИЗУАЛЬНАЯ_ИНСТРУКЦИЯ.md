## 📊 Структура данных

### Как это хранится в базе данных:

```
Restaurant (Ваш ресторан)
    ↓
Category (Категория: "Супы")
    ↓
Dish (Блюдо: "Борщ")
    ├── name: "Борщ"
    ├── description: "Традиционный украинский суп"
    ├── price: 1200
    └── imageUrl: "/uploads/dishes/borsch.jpg"
```

### Где хранятся файлы:

```
d:\QR MENU\
├── backend\
│   └── uploads\
│       └── dishes\
│           ├── borsch.jpg      ← Фото борща
│           ├── solyanka.jpg    ← Фото солянки
│           └── caesar.jpg      ← Фото цезаря
```

## 💡 Полезные советы

### 1. Порядок категорий

```
Супы        (sortOrder: 1)  ← Первая
Салаты      (sortOrder: 2)  ← Вторая
Горячие     (sortOrder: 3)  ← Третья
Десерты     (sortOrder: 4)  ← Четвертая
```

### 2. Качество фото

```
✅ ХОРОШО:
- Размер: 800x600 пикселей
- Формат: JPG или PNG
- Вес: 500KB - 2MB
- Качество: Четкое, яркое

❌ ПЛОХО:
- Размер: Слишком большое (>5MB)
- Формат: BMP, TIFF
- Качество: Размытое, темное
```

### 3. Описания блюд

```
✅ ХОРОШО:
"Традиционный украинский суп с говядиной,
свеклой, капустой и сметаной. Вес: 350г"

❌ ПЛОХО:
"Суп"
```

## 🐛 Частые проблемы

### Проблема 1: Не вижу кнопку "+ Блюдо"

```
❌ ВЫ ВИДИТЕ:
┌──────────────────────────────────┐
│  ▶  Супы                         │
│     0 блюд                       │
└──────────────────────────────────┘

✅ ДОЛЖНЫ ВИДЕТЬ:
┌──────────────────────────────────────────────┐
│  ▶  Супы                                     │
│     0 блюд                                   │
│                                              │
│              [+ Блюдо]  [✏️]  [🗑️]          │
└──────────────────────────────────────────────┘
```

**Решение:** Прокрутите вправо или увеличьте окно браузера

---

### Проблема 2: Блюдо не появляется

**Причина:** Категория свернута

```
❌ СВЕРНУТА (не видно блюд):
┌──────────────────────────────────┐
│  ▶  Супы                         │
│     1 блюдо                      │
└──────────────────────────────────┘

✅ РАЗВЕРНУТА (видно блюда):
┌──────────────────────────────────┐
│  ▼  Супы                         │
│     1 блюдо                      │
│     ┌────────────────────────┐   │
│     │  [фото]  Борщ          │   │
│     │          1200 ₸        │   │
│     └────────────────────────┘   │
└──────────────────────────────────┘
```

**Решение:** Нажмите на стрелку ▶ чтобы раскрыть

---

### Проблема 3: Фото не загружается

```
❌ ОШИБКА:
"Ошибка при загрузке изображения"

✅ ПРОВЕРЬТЕ:
- Размер файла < 5MB
- Формат: JPG, PNG, GIF, WEBP
- Папка существует: backend/uploads/dishes/
```

**Решение:**

```powershell
# Создайте папку
New-Item -ItemType Directory -Path "d:\QR MENU\backend\uploads\dishes" -Force
```
