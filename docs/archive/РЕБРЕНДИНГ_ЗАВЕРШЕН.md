## 🚀 Быстрый деплой (30 минут)

### Шаг 1: Supabase (5 мин)

```
1. supabase.com → New Project
2. Название: oimoqr-db
3. Скопировать DATABASE_URL
```

### Шаг 2: Render (10 мин)

```
1. render.com → New Web Service
2. Название: oimoqr-backend
3. GitHub: ваш-репозиторий
4. Environment variables
5. Deploy
```

### Шаг 3: Vercel (10 мин)

```
1. vercel.com → New Project
2. Название: oimoqr
3. VITE_API_URL
4. Deploy
5. Домены: oimoqr.com, *.oimoqr.com
```

### Шаг 4: DNS (5 мин)

